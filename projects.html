Research Projects

Denoising Diffusion Probabilistic Model for Retinal Image Generation and Segmentation.

2023 IEEE International Conference on Computational Photography (ICCP).

In this work, I developed a novel lightweight version of Denoising Diffusion Probabilistic Model (DDPM) that was used to generate a paired dataset for retinal image segmentation, containing over 30,000 image pairs. The changes to DDPM's architecture allowed the increase of resolution from 64×64 up to 512×512 on NVIDIA RTX 2080 SUPER. As a result, the proposed changes decreased the image generation time by a factor of 2 compared to the original DDPM, while having better Frechet Inception Distance scores.

A Hybrid Approach for retinal image super-resolution.

Biomedical engineering advances 6.

In this work, I proposed a novel fixed progressively growing super-resolution model that combines a vision transformer encoder and a convolutional neural network decoder. Another key feature is Adaptive Patch Embedding Layer that keeps the architecture unchanged when working with different image resolutions. The proposed model is lightweight allowing it to up-scale image up to 2048×2048.

Retinal Image Restoration using Transformer and Cycle-Consistent Generative Adversarial Network.

2022 International Symposium on Intelligent Signal Processing and Communication Systems (ISPACS).

In this work, a novel lightweight Vision Transformer-based CycleGAN has been made to restore unpaired low-quality retinal images. A Vision Transformer Encoder was embedded in the network that significantly improved the results.

Retinal Image Restoration and Vessel Segmentation using Modified Cycle-CBAM and CBAM-UNet.

2022 Innovations in Intelligent Systems and Applications Conference (ASYU).

In this work, a CycleGAN and UNet have been modified to accomodate Convolution Block Attention Module (CBAM) for unpaired retinal image restoration and vessel segmentation.
